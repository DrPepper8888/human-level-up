# Human-Level-Up 🚀

> **“如果 AI 学会了而你没学会，那你只是在给代码打工。”**

### 为什么需要它？
面对长文档，人类总是习惯点击“总结”，然后产生“我懂了”的幻觉。`Human-Level-Up` 强制打破这种幻觉。它像一个冷静的教官，先把整座森林指给你看，然后拎着你的耳朵带你爬最险的山头。

### 核心黑科技
- **认知地图 (Cognitive Map)**：不再被长篇大论淹没，先看清知识的脉络。
- **第一性原理拆解**：所有复杂名词都会被还原成“带泥土气息”的逻辑原子。
- **硬核命题**：不通过它的“脑力突袭”，它会一直用毒舌督促你思考，直到你的神经元真正完成重连。

### 触发指南
丢入长文、论文或代码仓库后，输入：
- **“从中能学到什么？”** —— 开启地图扫描
- **“开始进化！”** —— 接受逻辑洗礼

## 🛠 增强套件 (Scripts Bundle)
为了让本协议贴近研究生及高阶开发者需求，我们在 `/scripts` 目录提供以下辅助：
- **arXiv 燃料采集器 (`paper_miner.py`)**: 自动抓取前沿论文摘要，直接喂给教官，省去筛选时间。
- **长文档处理器 (`pre_processor.py`)**: 解决 Token 限制，确保大部头著作不丢失细节。
- **进化追踪器 (`tracker.py`)**: 自动提取学习成果与被毒舌的“战损记录”，生成周报。
